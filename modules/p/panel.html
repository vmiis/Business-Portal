Hover the mouse over the link to see where the module is hosted.
The modules can be hosted on any static web servers around the world and maintained by different people with different skills.
Each module's database access is fully controlled by the owner of the module's database table.
You can start to build your own application by clone this repository on GitHub (https://github.com/vmiis/business-portal) and connect each module to your own database table
